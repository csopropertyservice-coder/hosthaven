🔐 Security Overview — Hardware Key (YubiKey) Guide
Hardware security keys (YubiKey 5 Series, ~$55) provide phishing-resistant MFA. A stolen password alone cannot access your accounts. Set one up on all three platforms below.
▲
Vercel
Hosting
1. Go to vercel.com → Settings → Security
2. Under Two-Factor Authentication, click Enable
3. Choose Security Key (WebAuthn) — not TOTP
4. Insert your YubiKey and tap it when prompted
5. Save a backup recovery code in your password manager
✓ Protects: deployments, env vars, domain config
⚡
Supabase
Database
1. Go to supabase.com → Account Settings → Security
2. Under Multi-Factor Authentication, click Add a Factor
3. Supabase currently supports TOTP — use Authy or Google Authenticator
4. For physical key protection, secure your GitHub account (Supabase connects via GitHub OAuth)
5. GitHub: Settings → Password & auth → Security keys → Add
✓ Protects: database, RLS policies, edge functions, API keys
G
Business Gmail
Critical
1. Go to myaccount.google.com → Security → 2-Step Verification
2. Scroll to Security Keys → click Add security key
3. Insert YubiKey and follow prompts — it registers as a passkey
4. Set YubiKey as primary, phone as backup only
5. Under Advanced Protection Program — enroll if you handle contractor payments
✓ Protects: all password reset emails, Stripe notifications, contractor comms
⚠ Middleman Model Risk: Your Gmail is the single point of failure for contractor payments and owner communications. A phished Gmail = fraudulent wire instructions to your contractors. The YubiKey on Gmail is your highest-priority security action.
